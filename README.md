## Quick Start Guide

### Installation

```
# Clone the repository (use the Force, Luke!)
git clone https://github.com/Frytskyy/WireGuard-Star-Network-Manager-Configurator.git
cd WireGuard-Star-Network-Manager-Configurator

# Install dependencies
pip install qrcode[pil] cryptography

# Optional: Install WireGuard tools for enhanced key generation
# Ubuntu/Debian (the civilized way)
sudo apt install wireguard-tools

# CentOS/RHEL (for the masochists)  
sudo yum install wireguard-tools

# macOS (for the hipsters with money)
brew install wireguard-tools

# Run the script
python3 wireguard_config_gen_public.py
```

### First Steps

1. Create Your First Network: Choose the default network option for instant gratification
2. Add Nodes: Your devices, servers, IoT gadgets - everything gets a place in the galaxy
3. Generate Configs: Transform your network data into actual WireGuard configurations
4. Create QR Codes: For easy mobile device setup (your thumbs will thank you)
5. Deploy & Connect: Use the generated configs to bring balance to your network

## Network Architecture

This tool creates star topology networks where all nodes connect through a central server:

```
    [Mobile Device]          [Laptop]
         |                        |
         |                        |
    [Desktop] ---- [VPN Server] ---- [Workstation]  
         |                        |
         |                        |
    [IoT Hub]              [Gaming Rig]
```

Perfect for:
- Home Networks: Connect all your devices securely
- Small Business: Remote access for employees  
- Development: Isolated environments for testing
- IoT Management: Secure communication for smart devices
- Site-to-Site: Branch office connectivity

## Configuration Files

The tool generates standard WireGuard configurations:

### Server Config (wg0.conf)
```
[Interface]
Address = 192.168.100.1/32
PostUp = iptables -A FORWARD -i %i -j ACCEPT; iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE
PostDown = iptables -D FORWARD -i %i -j ACCEPT; iptables -t nat -D POSTROUTING -o eth0 -j MASQUERADE
ListenPort = 51820
PrivateKey = [SERVER_PRIVATE_KEY]

[Peer]
# Mobile Device
PublicKey = [CLIENT_PUBLIC_KEY]
AllowedIPs = 192.168.100.20/32
```

### Client Config
```
[Interface]
PrivateKey = [CLIENT_PRIVATE_KEY]
Address = 192.168.100.20/32
DNS = 192.168.100.1

[Peer]
PublicKey = [SERVER_PUBLIC_KEY]
AllowedIPs = 192.168.100.0/24
Endpoint = your-server.com:51820
PersistentKeepalive = 25
```

## Troubleshooting

### Common Issues

QR codes not generating?
```
pip install qrcode[pil]
```

Encryption not available?  
```
pip install cryptography
```

Key generation failing?
```
# Install WireGuard tools
sudo apt install wireguard-tools
```

Permission denied on server?
```
sudo chmod +x wireguard_config_gen_public.py
```
